# Adventure Aviator

Adventure Aviator is an autonomous flight school platform used here as proof of product craftsmanship and operational UI thinking. It pairs a strong thematic identity with production-oriented frontend choices, real-time data integration, and a continuous control mindset.

## What It Demonstrates

- end-to-end product design discipline
- responsive frontend implementation
- themed but usable user experience
- real-time operational data surfaces

## Why It Belongs In This Portfolio

Hiring managers often want to know whether an architect can also ship. Adventure Aviator answers that by showing that Ross can build polished interfaces that still feel intentional, usable, and operationally aware.
