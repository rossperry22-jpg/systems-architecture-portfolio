# Agent Model

Project Syndicate is built around `22` specialized agent roles.

## Coordination Principles

- each agent has a bounded responsibility
- orchestration logic routes work instead of collapsing all tasks into one model
- operator oversight remains visible and accessible
- secure API boundaries protect integration points

## Representative Role Groups

- research and discovery agents
- planning and synthesis agents
- execution and implementation agents
- quality and validation agents
- operator support agents

## Architecture Lesson

The deeper lesson is that role clarity is a systems problem, not just a staffing problem. The same pattern applies to sales engineering, solution design, and implementation planning.
